# 2026-06-03 邮箱验证码与密码重置功能

## 日期和时间
2026-06-03 12:15

## 修改内容
1. **邮箱验证码系统**：完整的邮箱验证码发送、校验、频率限制功能
2. **注册邮箱绑定**：注册时可选绑定邮箱，通过验证码验证邮箱归属
3. **密码重置**：通过邮箱验证码重置密码（无需登录）
4. **前端 UI 改造**：注册页新增邮箱字段和验证码交互，登录页新增"忘记密码"入口

## 修改原因
用户需要：
- 注册时绑定邮箱用于账号安全和密码找回
- 忘记密码时通过邮箱验证码重置密码
- 邮箱验证码的安全发送和校验机制

## 影响范围
- 认证系统（auth 模块）
- 数据库（SQLite 新增 email_verification_codes 表，users 表新增 email/email_verified 字段）

- SMTP 邮件服务配置

### 数据库设计

```sql
-- email_verification_codes 表
CREATE TABLE email_verification_codes (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    email TEXT NOT NULL,
    code TEXT NOT NULL,
    purpose TEXT NOT NULL,          -- register/reset_password/bind_email
    username TEXT,                  -- 关联用户名（注册时）
    attempt_count INTEGER DEFAULT 0,
    expires_at TEXT NOT NULL,
    used INTEGER DEFAULT 0,
    created_at TEXT NOT NULL
);
CREATE INDEX idx_email_codes_email ON email_verification_codes(email);
CREATE INDEX idx_email_codes_expires ON email_verification_codes(expires_at);
CREATE INDEX idx_email_codes_purpose ON email_verification_codes(purpose);

-- users 表新增字段
ALTER TABLE users ADD COLUMN email TEXT;
ALTER TABLE users ADD COLUMN email_verified INTEGER NOT NULL DEFAULT 0;
```

### 安全设计

| 防护措施 | 规则 |
|---------|------|
| 验证码生成 | `secrets.randbelow(1000000)` 6位数字 |
| 发送频率 | 同一邮箱 60秒内仅1次 |
| 每日上限 | 同一邮箱每天最多 10 次 |
| 有效期 | 5 分钟自动过期 |
| 暴力破解 | 最多验证 5 次，超过后验证码失效 |
| 邮箱唯一 | 注册时检查邮箱是否已被绑定 |
| 密码重置后 | 注销该账号全部会话 |

### HF Spaces 环境变量配置

需要在 HF Space Secrets 中配置：
- `SMTP_USER`：发件邮箱地址（如 QQ邮箱）
- `SMTP_PASSWORD`：邮箱授权码（非登录密码）

硬编码默认值：
- `SMTP_HOST=smtp.qq.com`
- `SMTP_PORT=465`
- `SMTP_USE_SSL=true`

## 测试方案

1. **注册邮箱绑定**：
   - 填写用户名、密码 → 输入邮箱 → 点击"发送验证码" → 查收邮件 → 输入验证码 → 点击"验证" → 显示"已验证" → 提交注册
   
2. **密码重置**：
   - 登录页点击"忘记密码？" → 输入邮箱 → 发送验证码 → 输入验证码 + 新密码 → 重置成功 → 使用新密码登录

3. **频率限制**：
   - 60秒内重复发送应提示"发送过于频繁"
   - 每日超过10次应提示"已达上限"

4. **安全验证**：
   - 验证码过期后校验应失败
   - 错误验证码5次后应失效
   - 密码重置后旧会话应全部注销
