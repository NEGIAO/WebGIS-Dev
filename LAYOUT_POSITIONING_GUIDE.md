# 用户中心与鹰眼视图位置配置

## 📍 当前布局

### 桌面设备 (默认)
```
鹰眼视图位置 (MapContainer.vue):
├─ 位置: left: 10px, top: 10px
├─ 尺寸: 200px × 200px
└─ 占用范围: x:[10-210], y:[10-210]

用户中心位置 (FloatingAccountPanel.vue):
├─ FAB 按钮: left: 220px, top: 20px
├─ 面板宽度: ~360px
└─ 不重叠范围: x:[220+], y:[20+]
```

### 移动设备 (@media max-width: 768px)
```
用户中心位置:
├─ FAB 按钮: left: 10px, top: 220px
├─ 面板宽度: 95vw (最大 420px)
└─ 显示在鹰眼下方，避免覆盖
```

## 🎯 位置计算逻辑

| 设备类型 | 鹰眼位置 | 用户中心位置 | 理由 |
|---------|---------|-----------|------|
| 桌面 | `left:10, top:10`<br>200×200px | `left:220, top:20`<br>鹰眼右侧 | 充分利用屏幕空间，不重叠 |
| 平板/手机 | `left:10, top:10`<br>200×200px | `left:10, top:220`<br>鹰眼下方 | 屏幕宽度有限，纵向排列 |

## 🔧 如需调整

如果需要改变位置间距，请修改这些值：

### 桌面设备间距调整
```css
.floating-account-manager {
  top: 20px;      /* ← 调整与页面顶部的距离 */
  left: 220px;    /* ← 调整鹰眼右侧的距离 (10+200+10) */
}
```

### 移动设备间距调整
```css
@media (max-width: 768px) {
  .floating-account-manager {
    top: 220px;    /* ← 调整与鹰眼下方的距离 (10+200+10) */
    left: 10px;    /* ← 调整与页面左侧的距离 */
  }
}
```

## 🎨 相关文件

- **MapContainer.vue**: 鹰眼视图样式 (`:deep(.ol-custom-overviewmap)`)
- **FloatingAccountPanel.vue**: 用户中心样式 (`.floating-account-manager`)

## 💡 为什么这样设计

1. **充分利用空间**: 桌面设备左上和右上分别放置两个组件
2. **移动友好**: 手机上垂直排列，避免水平拥挤
3. **无遮挡**: 两个组件完全分离，互不影响
4. **一致性**: 与地图控件布局规范一致
